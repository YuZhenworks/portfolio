蟲解食代形象網站

此專案是為了畢業專題自架一個形象網站，以網頁形式介紹理念、產品。

網頁的視覺設計希望不失蟲解食代的未來感，使用黑色背景搭配螢光綠元素點綴，凸顯產品本身的設計風格。

切版使用 Bootstrap 開發並使用 Sass/SCSS 管理專案，透過 jQuery 動態效果撰寫、插件套用，提升網頁互動效果。
