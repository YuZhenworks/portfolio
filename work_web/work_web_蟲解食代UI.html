蟲解食代形象網站

蟲解食代形象網站設計

此設計案例是替畢業專題自架一個形象網站，網頁的視覺設計希望不失專題的未來感，使用黑色背景搭配螢光綠元素點綴，凸顯產品本身的設計風格。

在首頁的設計上以一頁式網站為架構，並放上廣告影片幫助用戶在進入網站當下能快速理解核心內容。

產品頁以展示產品「蟲解能量棒」的各式包裝為主，並在下方製作說明小卡，用戶能透過點擊小卡翻面，背面完整介紹各個營養BUFF蟲的資訊及營養價值。

由於這是我第一次設計及架設完整的形象網站，當中碰到了許多挑戰，例如首頁的廣告影片在網站部署後手機端卻不能播放，或是說明小卡在翻面的過程中有圖片跑圖等問題。

為了更加成功地介紹蟲解食代的設計理念及衍生產品，通過選擇適合的元件及圖形元素，設計了一個簡潔、直觀的網站介面，以最大程度地展示主題內容，最終透過不斷的測試除錯與設計協調，找到了最佳的解決方案。
